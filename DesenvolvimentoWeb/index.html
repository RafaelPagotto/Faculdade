🎯 O que é Programação Web?

Programação web é a criação de páginas e aplicações que funcionam nos navegadores. É composta principalmente por três tecnologias:

  • HTML - Estrutura e conteúdo
  • CSS - Apresentação e estilo
  • JavaScript - Interatividade e lógica

📄 HTML - HyperText Markup Language

O que é HTML?

HTML é a linguagem de marcação que estrutura o conteúdo da web. Ela usa tags (etiquetas) para definir diferentes tipos de conteúdo.

Exemplo 1: Estrutura Básica

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Minha Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é meu primeiro parágrafo.</p>
</body>
</html>

📝 Anotação: Tags Comuns

  • <h1> a <h6> - Títulos (hierarquia)
  • <p> - Parágrafos
  • <div> - Container genérico
  • <a> - Links (âncoras)
  • <img> - Imagens
  • <ul>/<ol> - Listas

🎨 CSS - Cascading Style Sheets

O que é CSS?

CSS é a linguagem que define a aparência visual das páginas web. Ela controla cores, fontes, layouts, espaçamentos e muito mais.

Exemplo 3: Sintaxe CSS

/* Seletor - Define o que estilizar */
h1 {
    color: #2c3e50;           /* Cor do texto */
    font-size: 2.5rem;        /* Tamanho da fonte */
    text-align: center;       /* Alinhamento */
    margin-bottom: 1rem;      /* Espaço externo */
}

/* Classes - Reutilizáveis */
.destaque {
    background-color: #f39c12;
    padding: 10px;
    border-radius: 5px;
}

📝 Anotação: Três Formas de Aplicar CSS

  1. Inline: <p style="color: red;"> - Direto no elemento
  2. Interno: <style> dentro do <head>
  3. Externo: <link rel="stylesheet" href="styles.css"> - Arquivo separado (recomendado!) ✅

📝 Anotação: Layouts Modernos
